# UIKit Track

Use this track for UIKit-first apps, legacy codebases, or teams with minimal SwiftUI usage.

- Backlog: `docs/uikit/guidelines/topic-backlog.md`
- Guidelines: `docs/uikit/guidelines/`
- Shared standards: `docs/core/`

## Track Status

- Guideline set `U-001` through `U-015` is accepted.
- Backlog rows include:
  - `status`
  - `acceptance_status`
  - `review_evidence`
  - `citation_tiers`

## How To Use

1. Pick a topic from the backlog.
2. Open the corresponding `u-xxx` guideline file.
3. Apply guidance with Tier-1-backed citations and testing sections.
