# References Map

This folder is the skill-facing reference tree used by `SKILL.md`.

## Structure

- `manifests/` skill runtime routing and decision manifests
- `core/` shared accessibility standards and templates
- `swiftui/` SwiftUI backlog and guidelines
- `uikit/` UIKit backlog and guidelines
- `tvos/` tvOS backlog and guidelines
- `macos/` macOS backlog and guidelines
- `visionos/` visionOS backlog and guidelines
- `testing/` verification planning documents

## Note

These references mirror project docs/manifests to make skill installation and discovery deterministic.
